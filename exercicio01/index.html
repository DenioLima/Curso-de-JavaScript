Primeiro exercicio de JavaScript

Para usar JavaScript vou sempre usar a tag Script no fim do body ou abrir uma pagina separada so para ele, como se faz no css.

Vou usar o alert para emitir um alerta, se quiser uma confirmação vou usar o confirm, prompt para fazer uma pergunta. Vou usar o var q é variavel e vou dar o nome de maior para quardar a resposta do prompt. E depois vou usar o alert novamente para dar a resposta q esta na minha variavel, e vou usar o sinal de + para unir a frase com a variavel no alert.
